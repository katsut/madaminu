$players_info

## タスク

事件発生時に各プレイヤーが持っている証拠またはアリバイを生成してください。
各プレイヤーにちょうど2つずつ配布します。

### ルール
- 各アイテムは「evidence（証拠）」か「alibi（アリバイ）」のどちらか
- **アリバイの整合性**: アリバイが「AさんとBさんが一緒にいた」という内容なら、AもBも同じアリバイを持つこと
- 証拠は個人的な発見（目撃情報、物的証拠など）
- 犯人のアイテムは、犯人にとって不利にも有利にもなり得る曖昧なもの
- アリバイは必ず別のプレイヤーとペアになる

### 出力形式

```json
{
  "items": [
    {
      "type": "evidence",
      "owner_ids": ["player_id_1"],
      "title": "証拠のタイトル",
      "content": "証拠の内容（2〜3文）"
    },
    {
      "type": "alibi",
      "owner_ids": ["player_id_1", "player_id_2"],
      "title": "アリバイのタイトル",
      "content": "アリバイの内容（例: 事件発生時にAとBは庭で話していた）"
    }
  ]
}
```

### ガイドライン
- 各プレイヤーがちょうど2つのアイテムを受け取るように配分する
- アリバイは2人のプレイヤーで共有されるので、1つのアリバイで2人分カウント
- 犯人には偽のアリバイ（実は成立しない）や、犯行とは別の疑わしい行動の証拠を持たせる
- 目撃者には犯行に関連する目撃証拠を持たせる
- マップの場所を具体的に参照して空間的な整合性を持たせる
