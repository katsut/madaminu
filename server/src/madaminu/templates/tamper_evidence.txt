## 証拠のすり替え

プレイヤー「$player_name」が「$location_name」で発見した証拠を、無関係なものにすり替えます。

### 元の証拠
- タイトル: $original_title
- 内容: $original_content

## タスク

この証拠を、事件とは無関係に見える別のものに書き換えてください。
同じ場所にあっても不自然ではないが、事件の手がかりにはならないものにします。

```json
{
  "title": "すり替え後のタイトル（日本語、簡潔に）",
  "content": "すり替え後の描写（日本語、2〜3文。元の場所にあっても不自然でないもの）"
}
```

### ガイドライン
- 元の証拠と同じ場所・同じ種類のものだが、事件とは無関係な内容にする
- 例: 「血のついた手紙」→「古い広告チラシ」、「割れた花瓶の破片」→「装飾用の石」
- 自然に見えるすり替えにする（他のプレイヤーが後で調べても不自然に思わない程度）
