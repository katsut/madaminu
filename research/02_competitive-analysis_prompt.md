# マーダーミステリーアプリ・サービスの競合分析と市場調査

```
オンラインマーダーミステリーゲームの競合分析と市場動向について包括的に調査してください。

## 調査目的
4〜7人がリアルタイムで参加し、LLMによるシナリオ自動生成とキャラクター作成機能を持つiOSマーダーミステリーアプリを開発するにあたり、既存の競合製品と市場環境を把握したい。

## 調査項目

### 1. 既存のオンラインマーダーミステリーアプリ・サービス
以下の各サービスについて調査してください:
- アプリ名、プラットフォーム（iOS/Android/Web/Steam等）
- ゲームの形式（テキスト、音声、ビデオ、カード等）
- プレイヤー人数
- シナリオの提供方式（固定シナリオ、ユーザー作成、AI生成等）
- 料金体系（無料/有料/サブスクリプション）
- ユーザー評価とレビュー数
- 主要な強みと弱み
- 日本語対応の有無

特に以下のカテゴリのサービスを網羅してください:
- 日本のマーダーミステリーアプリ（ウズ、マダミス.jp など）
- 海外のマーダーミステリーアプリ（Among Us類似、専用アプリ）
- AI/LLMを活用したインタラクティブ推理ゲーム
- テーブルトップRPG系（Roll20、FoundryVTT等でのマダミスプレイ）
- マーダーミステリーイベント主催サービス（オフラインだがデジタルツール併用）

### 2. AI/LLMを活用した類似ゲーム
- AI Dungeonなどのインタラクティブフィクション
- LLMを使った推理ゲームや物語生成ゲームの事例
- ChatGPTやClaudeをマーダーミステリーのGM（ゲームマスター）として使う事例
- AIキャラクターとの対話型推理ゲーム

### 3. 市場規模とトレンド
- 日本のマーダーミステリー市場の規模と成長率（2020年〜現在）
- 世界のマーダーミステリー/推理ゲーム市場の規模
- オンラインマーダーミステリーの普及状況（COVID-19以降の変化）
- モバイルゲーム市場におけるソーシャル推理ゲームの位置付け
- 日本でのマーダーミステリーブームの背景と今後の展望

### 4. ユーザー層・プレイヤー特性
- マーダーミステリーのプレイヤー層（年齢、性別、プレイ頻度）
- 初心者とヘビーユーザーのニーズの違い
- オンラインプレイとオフラインプレイの選好
- 日本と海外のプレイヤー文化の違い

### 5. 差別化ポイントの特定
- 既存サービスで満たされていないニーズ
- 「キャラクター作成→AI自動生成シナリオ→リアルタイム進行→結末未確定」というコンセプトの新規性
- 競合優位性を確立するためのポイント

## 出力形式
- 各サービスの情報は表形式で比較してください
- 市場データには出典を明記してください
- 強み・弱みの分析にはSWOTフレームワークを使用してください
- 差別化ポイントは具体的なアクションに落とし込んでください
```
