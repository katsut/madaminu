# マダミヌ ゲームルール

## 概要

マダミヌは、AIが生成するオリジナルシナリオで遊ぶマーダーミステリーゲームです。4人以上のプレイヤーが集まり、事件の真相を推理します。プレイヤーの1人が犯人ですが、誰が犯人かはAIが決定します。

---

## ゲームの流れ

### 1. 準備

- ホストがルームを作成し、他のプレイヤーが参加
- 各プレイヤーがキャラクターを作成（名前・性別・年齢・職業・外見・性格・経歴）
- 4人に満たない場合、AIプレイヤーが自動補充される
- ホストが「ゲーム開始」を押すと、AIがシナリオを生成

### 2. シナリオ配布

ゲーム開始後、各プレイヤーに以下が配られます：

- **事件の舞台と状況** — 全員共通
- **被害者の情報** — 全員共通
- **自分の秘密情報** — 自分だけが知っている事実
- **自分の目的** — ゲーム中に達成すべき個人目標
- **自分の役割** — 犯人・目撃者・関係者・無実のいずれか（自分だけに表示）

### 3. 調査フェーズ

- 画面に表示される**場所**を選んで調査できます
- 1フェーズにつき**最大3回**まで調査可能
- 調査するとAIが証拠や手がかりを生成し、自分の手帳に記録されます
- 発言ボタンを押して他のプレイヤーと情報を共有することもできます

### 4. 議論フェーズ

- **発言ボタン**を押して発言権を取得します（同時に話せるのは1人だけ）
- 集めた証拠をもとに推理を述べ、他のプレイヤーと議論します
- 犯人は自分が犯人だとバレないよう立ち回りましょう
- 発言が終わったら「発言終了」ボタンで発言権を返します

### 5. 投票フェーズ

- 犯人だと思うプレイヤーに投票します
- 全員が投票を完了するとエンディングに進みます

### 6. エンディング

- **真犯人**の正体が明かされます
- 事件の真相（動機やトリック）が語られます
- 各プレイヤーの**個人目的の達成状況**が発表されます

---

## 役割

| 役割 | 説明 |
|------|------|
| **犯人** | 事件の犯人。正体を隠しつつ、個人目的の達成を目指す |
| **目撃者** | 重要な情報を持っている。真実を明かすか隠すかはあなた次第 |
| **関係者** | 事件に何らかの関わりがある。秘密を守りつつ推理に参加 |
| **無実** | 事件とは直接関係ない。純粋に推理を楽しめる |

---

## ルールのポイント

- **秘密情報は自分だけのもの** — 他のプレイヤーには見えません。共有するかどうかはあなたの判断
- **嘘をついてもOK** — マーダーミステリーでは嘘も戦略の一部です
- **発言は1人ずつ** — 発言ボタンで順番に話します
- **手帳を活用** — 調査で得た証拠やGMからの手がかりは手帳に自動記録されます
- **制限時間あり** — 各フェーズには制限時間があります。ホストは延長や次フェーズへの進行が可能です

---

## 勝利条件

マダミヌには明確な勝敗はありませんが、以下が目標です：

- **犯人以外** → 真犯人を見つけて投票で指名する
- **犯人** → 正体がバレないよう振る舞い、他の人に投票を向けさせる
- **全員** → 自分の個人目的を達成する
