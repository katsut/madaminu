# マダミヌ ゲームルール

## 概要

マダミヌは、AIが生成するオリジナルシナリオで遊ぶマーダーミステリーゲームです。4人以上のプレイヤーが集まり、事件の真相を推理します。プレイヤーの1人が犯人ですが、誰が犯人かはAIが決定します。

---

## ゲームの流れ

### 1. 準備

- ホストがルームを作成（ルーム名・パスワード任意）
- 他のプレイヤーがルームに参加
- 各プレイヤーがキャラクターを作成（名前・性別・年齢・職業・外見・性格・経歴）
- 4人に満たない場合、AIプレイヤーが自動補充される（シナリオに合ったキャラクター）
- 全員が「準備完了」を押したら、ホストが「ゲーム開始」

### 2. シナリオ配布（イントロ）

ゲーム開始後、各プレイヤーに以下が配られます：

- **事件の舞台と状況** — 全員共通
- **被害者の情報** — 全員共通
- **自分の公開情報** — 全員に見える自分の立場（誰と来たか、被害者との関係等）
- **自分の秘密情報** — 自分だけが知っている事実
- **自分の目的** — ゲーム中に達成すべき個人目標（※他のプレイヤーにバレてはいけない）
- **初期証拠とアリバイ** — 事件発生時に持っていた証拠カード1枚とアリバイカード1枚
- **自分の役割** — 犯人・目撃者・関係者・無実のいずれか（自分だけに表示）

全ページを確認し、全員が「準備完了」を押したらゲーム本編が始まります。

### 3. 自己紹介フェーズ（5分）

- 発言ボタンを押して、自分のキャラクターを紹介
- この集まりに来た理由、他のキャラクターとの関係を共有
- 推理はまだしない。まずはお互いを知る時間

### 4. 調査サイクル（3ターン繰り返し）

各ターンは以下の3つのフェーズで構成されます：

#### 4a. 調査計画フェーズ（3分）

- マップを見て、次に調べたい場所を1つ選ぶ
- 発言ボタンで他のプレイヤーと相談できる
- 場所を選ばなかった場合、ランダムに割り当てられる
- 制限時間後、選んだ場所での調査が始まる

#### 4b. 調査フェーズ（2分）

- 選んだ場所にある「特徴」（家具、物品等）がリストで表示される
- 各特徴を調べるとAIが手がかりを生成
- 発見物から**1つだけ持ち帰れる**（証拠カードとして手帳に追加）
- 同じ場所に誰もいなければ、発見物を1つ**すり替える**（改ざん）ことも可能
- 同じ場所にいるプレイヤー同士はヒソヒソチャットができる

#### 4c. 議論フェーズ（5分）

- 発言ボタンを押して議論する
- 手持ちの**証拠カードを公開**できる（3ポイント）
- **口頭の主張**は1ポイント
- 他の人が発言中でも、発言ボタンを押すと割り込める
- 手帳を開きながら発言できる
- 議論の内容は手帳の「議論」タブに自動記録される

### 5. 投票フェーズ（制限時間なし）

- 犯人だと思うプレイヤーに投票
- 投票中も発言して最後の主張ができる
- **全員が投票を完了するまで待つ**（タイマーで自動進行しない）

### 6. エンディング

投票完了後、ドラマチックなリビール演出：

1. **「〇〇は・・・・」** — 最多票のキャラクターのアバターが表示される（4秒）
2. **「犯人でした」or「冤罪でした」** — 判定が大きく表示される（5秒）
3. **投票結果** — 誰が誰に投票したか。最多票のキャラクターが「監禁」される
4. **エピローグ** — AIが生成する結末の物語
   - 犯人が当たった場合: 告白、動機、トリックの説明
   - 冤罪の場合: 後味の悪い結末。真犯人は逃走する
5. **最終スコア** — ランキング形式で発表
6. **個人目的の達成状況** — 各プレイヤーの目標達成/未達成
7. **ネタバラシ** — 全キャラクターの秘密と役割が公開される

「もう一度見る」ボタンでリビール演出をリプレイできます。

---

## 役割

| 役割 | 説明 |
|------|------|
| **犯人** | 事件の犯人。正体を隠しつつ、個人目的の達成を目指す |
| **目撃者** | 重要な情報を持っている。真実を明かすか隠すかはあなた次第 |
| **関係者** | 事件に何らかの関わりがある。秘密を守りつつ推理に参加 |
| **無実** | 事件とは直接関係ない。純粋に推理を楽しめる |

---

## スコアリング

| アクション | ポイント |
|-----------|---------|
| 口頭での発言 | 1pt / 回 |
| 証拠カードの公開 | 3pt / 件 |

エンディングでランキング形式（🥇🥈🥉）で発表されます。

---

## 手帳

ゲーム中いつでも手帳を開けます（手帳を開きながら発言も可能）。

| タブ | 内容 |
|------|------|
| 自分 | 役割・秘密・目的・立場 |
| 登場人物 | 全キャラのプロフィール・アバター |
| マップ | 間取り図 |
| 証拠 | 収集した証拠カード一覧 |
| 議論 | ターンごとの発言・証拠提出の記録 |
| メモ | 自由記入欄 |

---

## ルールのポイント

- **秘密情報は自分だけのもの** — 他のプレイヤーには見えません。共有するかどうかはあなたの判断
- **個人目的はバレてはいけない** — 目的を達成しつつ、他人に気づかれないように
- **嘘をついてもOK** — マーダーミステリーでは嘘も戦略の一部です
- **証拠のすり替えが可能** — 単独で調査中なら、発見物を改ざんできる
- **制限時間あり** — 各フェーズには制限時間があります。ホストは延長・一時停止・次フェーズ進行が可能
- **再接続可能** — アプリが落ちてもルームに復帰できます

---

## ホスト機能

| 操作 | 説明 |
|------|------|
| フェーズを進める | 制限時間を待たずに次フェーズへ |
| フェーズを延長 | +60秒 |
| 一時停止 / 再開 | タイマーを停止・再開 |

---

## 勝利条件

マダミヌには明確な勝敗はありませんが、以下が目標です：

- **犯人以外** → 真犯人を見つけて投票で指名する
- **犯人** → 正体がバレないよう振る舞い、他の人に投票を向けさせる
- **全員** → 自分の個人目的を達成する
- **全員** → スコアを稼ぐ（発言 + 証拠提出）

---

## プレイ時間の目安

| フェーズ | 時間 |
|---------|------|
| 自己紹介 | 5分 |
| 調査計画 × 3 | 9分 |
| 調査 × 3 | 6分 |
| 議論 × 3 | 15分 |
| 投票 | 3分+ |
| **合計** | **約40分** |
